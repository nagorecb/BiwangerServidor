Descripción detallada

clase de lógica de negocio de usuario

Esta clase realiza todas las acciones de negocio relativas a la parte de "usuario", es decir, aquí es donde se da funcionalidad a los servicios de la parte usuario del programa. es aquí donde se realizan las acciones solicitadas por la parte cliente, siendo la capa que accede a la base de datos a través de DAO
